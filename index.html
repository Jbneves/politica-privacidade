Política de Privacidade – Currículo Fácil

Última atualização: 15/05/2025

Sua privacidade é importante para nós. Esta Política de Privacidade descreve como o aplicativo Currículo Fácil coleta, usa e protege as informações fornecidas pelos usuários.

1. Informações Coletadas

O aplicativo pode coletar os seguintes dados inseridos diretamente pelo usuário:

  • Nome completo
  • E-mail
  • Número de telefone
  • Objetivo profissional
  • Formação acadêmica
  • Experiências profissionais
  • Habilidades, idiomas e cursos
  • Links de redes sociais (ex: LinkedIn, GitHub)
  • Resumo profissional
  • Imagem da selfie (opcional)

Essas informações são utilizadas exclusivamente para gerar um arquivo PDF com o currículo do usuário. Nenhuma dessas informações é enviada a servidores externos ou armazenada fora do dispositivo.

2. Permissões Utilizadas

O app solicita as seguintes permissões:

  • Câmera: para capturar uma foto (selfie) que será incluída no currículo.
  • Armazenamento: para salvar o PDF gerado no dispositivo.
  • Galeria/Imagens: para permitir a seleção de uma imagem já existente.

Essas permissões são utilizadas somente para a funcionalidade descrita e não são usadas para coleta de dados não autorizada.

3. Geração e Compartilhamento de PDF

O currículo gerado é salvo localmente no seu dispositivo e pode ser compartilhado por você via e-mail, WhatsApp, ou outros aplicativos. Nenhum conteúdo é enviado automaticamente pelo app.

4. Publicidade

Este app utiliza o Google AdMob para exibir anúncios (banners e anúncios recompensados). O AdMob pode coletar dados, como:

  • Identificador do dispositivo
  • Interações com os anúncios
  • Informações técnicas do aparelho (modelo, sistema operacional, etc.)

Para mais detalhes, consulte a Política de Privacidade do Google.

5. Segurança

O app não armazena dados em servidores. Todos os dados são manipulados apenas em memória ou armazenados localmente no dispositivo para geração do currículo.

6. Compartilhamento de Dados

Nenhum dado pessoal é compartilhado com terceiros. O único serviço de terceiros utilizado é o Google AdMob, para fins de monetização via anúncios.

7. Consentimento

Ao utilizar o Currículo Fácil, você concorda com esta Política de Privacidade e com o uso das informações conforme descrito.

8. Alterações na Política

Esta Política de Privacidade pode ser atualizada periodicamente. Notificaremos os usuários sempre que houver alterações relevantes.
